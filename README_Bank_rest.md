# Bank Cards Management System

REST API сервис для управления банковскими картами с разделением ролей **USER** и **ADMIN**.

## Возможности

### Для пользователей:

- Регистрация и авторизация (JWT + Refresh Token)
- Выпуск новых карт
- Просмотр своих карт (с пагинацией)
- Запрос на блокировку карты
- Переводы между своими картами
- Просмотр полного номера карты (reveal PAN)

### Для администраторов:

- Полное управление картами (блокировка, разблокировка, удаление)
- Управление пользователями (блокировка, смена ролей)
- Управление продуктов карт (Classic, Gold, Platinum и т.д.)
- Обработка заявок пользователей на блокировку

## 🛠 Технологии

- **Java 21**
- **Spring Boot 3.2.4**
- **Spring Security + JWT**
- **Spring Data JPA**
- **PostgreSQL**
- **Liquibase** (миграции)
- **MapStruct**
- **Swagger / OpenAPI**
- **Docker Compose**

## Быстрый запуск

### 1. Через Docker (рекомендуется)

```bash
maven clean package
docker-compose up -d
```

После запуска приложения перейдите по этому url: http://localhost:8080/swagger-ui/index.html
